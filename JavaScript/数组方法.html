    # 操作数组元素方法

    # push()

    在数组尾部推入 数组元素, push()执行后 返回数组长度

    # pop()

    在数组中取得 最后一项元素, pop()执行后 返回该数组 最后一项 并且这一项从数组中删除, 数组长度也随之改变

    # shift()

    在数组中取得 第一项元素, shift()执行后 返回该数组 第一项 并且这一项从数组中删除, 数组长度也随之改变

    # unshift()

    在数组头部推入 数组元素, unshift()执行后 返回数组长度

    对数组操作时, 推入(向数组插入元素)元素 总返回该数组的长度, 取得(从数组删除元素) 总返回取得的这个数组元素

    # 栈方法(LIFO,后进先出)

    可通过push()和pop()实现

    # 队列方法(FIFO,先进先出)

    可通过shift()和push()实现

    # 排序方法

    # reverse()

    数组元素反转 var arr = [1,3.2,5.4] arr.reverse(); //[4,5,2,3,1]

    # sort()

    数组排序, sort中自定义比较函数确定排序方法, 比较函数返回值通过正数,负数,或0来确定最终排序 比较函数接收两个参数(a,b), 若a-b<0, 则a在b之前, 否则a在b之后; 若a-b==0那么这两个参数相等

    操作方法

    # concat()

    数组合并, 不改变原数组

    # slice()

    截取数组中的部分元素, 形成新数组, 和concat()一样不改变原数组, slice() 函数接收两个参数 slice(a, b), a表示起始位置, b表示结束位置( 但是新数组不会包含结束位置 对应的数组元素 )

    # splice()

    有很多用法, 可以删除, 插入, 替换数组元素, splice()会改变原数组 删除: 提供2个参数splice(a, b), a表示元素的索引(即 要从数组的什么位置开始删除), b表示要删除的数量

    插入: 提供3个参数splice(a, 0, c), a表示起始位置(数组索引), 0表示要删除的数量(不删除,再原数组元素不变的情况下 插入数组), c表示要插入的元素, 如果要插入多个值可以继续传参

替换: 提供3个参数splice(a, b, c), a表示起始位置(数组索引), b表示要删除的数量(可以删除任意数量的元素), c表示要插入的元素, 如果要插入多个值可以继续传参

    # 位置方法

    # indexOf()

    接收两个参数, 要查找的项和(可选的)查找其实位置的索引,从数组开头(位置0)向后查找

    # lastIndexOf()

    接收两个参数, 要查找的项和(可选的)查找其实位置的索引,从数组末尾开始向前查账

    # 迭代方法

    5个迭代方法, 每个方法都接收两个参数, 要在每一项上运行的函数和(可选的)运行该函数的作用域对象——影响this的值 传入这些方法中的函数会接收三个参数：数组项的值、该项在数组中的位置 和 数组对象本身

    # every()

    对数组中的每一项运行给定函数, 如果该函数对每一项都返回 true, 则返回 true

    # some()

    对数组中的每一项运行给定函数, 如果该函数对任意一项返回 true, 则返回 true

    # filter()

    对数组中的每一项运行给定函数, 返回该函数会返回true的项组成的数组

    # map()

    对数组中的每一项运行给定函数, 返回每次函数调用的结果组成的数组

    # forEach()

    对数组中的每一项运行给定函数, 这个方法没有返回值

    在这5个函数中every(), some()返回Boolean值, filter(), map()返回数组, forEach()没有返回值

    如果要将 数组中的全部元素 与 某一值作比较, 那么every(),some()是及其合适的, 不同的是 every()函数 只有全部元素比对后 都为true时 返回true(与&&类似) some()函数 全部元素比对后 有一项元素为true时 返回true(与||类似)

    filter()函数更像是过滤功能, 可以取出数组中某些符合条件的元素, filter()会返回一个新的数组

    map()函数则会对数组的所有元素生效 即 map()可以对函数的所有元素做相同操作

    forEach()函数本质上与for循环遍历数组时一样的

    var numbers2 = [1,53,5,13,5,8,45];
numbers2.forEach((item,index,arr) => {
    if(item>10) {
        arr[index] = item * 2;
    }
});
console.log(numbers2);
//item在forEach函数中,值始终为numbers2数组元素的初始值

    (思考遍历数组时,用forEach()替代for循环是否在所有场景下都适合?)

    # 归并方法

    接收4个参数, 前一个值、当前值、项的索引和数组对象 (prev, cur, index, array) 这个函数返回的任何值都会作为第一个参数自动传给下一项。第一次迭代发生在数组的第二项上，因此第 一个参数是数组的第一项，第二个参数就是数组的第二项

    # reduce()

    从第一项遍历到最后一项

    # reduceRight()

    从最后一项遍历到第一项
